[ 02 ] Transcriptomics

Bulk RNA-seq for expression & pathway analysis.

Transcriptome-wide profiling to identify differentially expressed genes, novel splicing events, and activated molecular pathways across experimental conditions.

Rigorous preprocessing
Quality inspection and automated adapter trimming using fastp, FastQC, and MultiQC.
Ultra-fast alignment
Spliced mapping using STAR/HISAT2 and alignment-free quantification using Salmon.
Differential expression
Statistical modeling of gene expression changes using DESeq2 and edgeR.
Functional pathway enrichment
Biological network analysis using KEGG, Gene Ontology (GO), and Reactome pathways.
STARHighly accurate spliced aligner for RNA-seq reads. SalmonTranscript-level quantification tool. DESeq2Statistical method for differential gene expression analysis. edgeREmpirical analysis of digital gene expression in R. ReactomeCurated biological pathway database.

[ 03 ] Single-Cell

scRNA-seq — cellular heterogeneity unlocked.

Profile transcriptomes at single-cell resolution. Identify rare cell populations, map developmental lineages, and characterize cellular microenvironments.

Clustering & cell-type annotation
Dimensionality reduction (PCA, UMAP, t-SNE) and automated cell classification.
Trajectory & pseudotime mapping
Developmental lineage tracking and RNA velocity computations.
Batch correction & integration
Multi-sample cohort integration utilizing Harmony, Seurat Integration, and MNN.
SeuratR toolkit for single-cell genomics. ScanpyPython-based single-cell analysis suite. Cell Ranger10x Genomics raw processing pipeline. HarmonyFast, robust batch effect correction algorithm. scVeloRNA velocity analysis tool.
